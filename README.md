# instagram-clone-django

> 취미로 시작한 인스타그램 클론 코딩입니다 :)  

![](./preview-profile.png)

## 기술스택
  - 서버 - AWS EC2
  - 형상관리 - GIT
  - language - Python 3.X
  - Web F/W - Django
  - Database - sqlite3

## 기능
  - 회원
    - 가입, 로그인, 로그아웃
  - 게시글
    - 조회, 등록, 수정, 삭제
    - 이미지 업로드
    - my profile
  - follower, following
  - like
  - book mark
  - comment
  - search
  - activity history
